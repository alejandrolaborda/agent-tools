# apple-tv-dev

Apple TV/tvOS development plugin with specialized agents for games and media apps.

## Installation

```bash
/plugin install apple-tv-dev@agent-tools
```

Requires: `apple-shared` plugin

## Agents

| Agent | Role |
|-------|------|
| `developer` | Code architecture and implementation |
| `designer` | UI/UX and Human Interface Guidelines |
| `qa-tester` | Testing, automation, TestFlight |
| `publisher` | App Store submission workflow |
| `legal` | Privacy, compliance, IP |
| `media` | Video/audio implementation |

## Skills

| Skill | Description |
|-------|-------------|
| `swiftui-tvos` | SwiftUI with focus system for tvOS |
| `focus-engine` | tvOS navigation and remote control |
| `avkit-media` | Video/audio playback, HLS streaming |
| `gamekit` | Game Center integration |
| `metal-graphics` | High-performance graphics |
| `testflight-qa` | Testing workflow |

## Usage

Use agents for specialized tasks:
```
@developer implement the game menu with focus navigation
@designer review the UI against tvOS HIG
@qa-tester set up XCUITest for remote control
```

## tvOS Version

Targets tvOS 18+ with Xcode 16+
