# apple-shared

Shared Apple platform skills for iOS, macOS, tvOS, watchOS, and visionOS development.

## Installation

```bash
/plugin install apple-shared@agent-tools
```

## Skills

| Skill | Description |
|-------|-------------|
| `swift-language` | Swift 6.2 language patterns, concurrency, memory management |
| `app-store-review` | App Store Review Guidelines compliance |
| `app-store-connect` | Publishing workflow and metadata |
| `accessibility` | VoiceOver, accessibility APIs |
| `localization` | Internationalization and localization |

## Usage

Skills activate automatically based on context. Mention Swift, App Store, or accessibility topics in your prompts.

## Dependencies

None - this is a foundational plugin.
